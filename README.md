# GJSERVER

GJSERVER Version 1.0

# HOW TO RUN

1. Download the source code
2. Download node.js for your platform (Windows/Linux/Other)
3. download the `ws`, `http`, `url` and `fs` packages
4. run `node server.js` in the same directory as the server.js file
5. connect to your server using the IP and PORT you has set
6. Done!!
